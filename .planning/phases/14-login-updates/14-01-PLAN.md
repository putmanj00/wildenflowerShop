# Plan 14.1: Forgot Password Flow
Implement a "Forgot Password" capability on the login screen, calling Shopify's customerRecover mutation.

## Steps
1. Add `CUSTOMER_RECOVER_MUTATION` to `lib/shopify-queries.ts`.
2. Add `recoverCustomer` function to `lib/shopify-auth.ts`.
3. Create `ForgotPasswordScreen` component at `app/(auth)/forgot-password.tsx`.
4. Link `app/(auth)/login.tsx` to `app/(auth)/forgot-password.tsx`.
5. Style the new screen matching the Botanical design system.

## Verification
- Enter email and tap "Reset Password" on the new screen.
- Verify success message ("If an account exists, an email will be sent...").
- Verify the Shopify `customerRecover` API call using the network inspector.
